# Webmin

Webmin és una eina d'administració de sistemes basada en web per a servidors tipus Unix, i els seus serveis.   
Utilitzant-lo, és possible fer configuracions interns del sistema operatiu, com ara usuaris, quotes de disc, serveis o fitxers de configuració, així com modificar i controlar aplicacions de codi obert, com ara BIND DNS Server, Apache HTTP Server, PHP, MySQL i molts més.  

## Abans de res
sudo apt update -y && sudo apt install git  
git clone https://github.com/capbussat/isard-scripts  
cd isard-scripts  
cd servidor-webmin  
sudo chmod +x servidor-webmin  
./servidor-webmin
